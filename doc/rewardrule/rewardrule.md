1. 연속 출석 보상
    
    
    | 연속 일수 | 지급 풀 |
    | --- | --- |
    | 1일 | 3 |
    | 2일 | 4 |
    | 3일 | 5 |
    | 4일 이상 | 5 (고정) |
    - 매일 00:00 기준 초기화
    - 연속 출석 끊기면 1일차부터 다시 시작
2. 배움 탭 보상
    - 문제 1개 정답: +1 풀
    - 3문제 세트 완료 시: +2 풀 추가 지급
    - 총 지급:
        - 1세트 최대 5 풀
3. 마음 탭 보상
    - 시나리오 1개 완료 시: +3 풀
4. 할 일 완료
    - 할 일 하나 완료 : +1 풀
    - 할 일 목록 다 끝냈을 경우 : + 3풀
5. 일일 기대 보상량
    - 배움 1세션: 약 5 풀
    - 마음 1회: 약 3 풀
    - 출석: 3~5 풀
    - 총합:
        - 약 10~15 풀 / 일
6. 레벨 시스템 
    - 레벨업 요구치는 선형 증가 (+5씩)
    
    | 레벨 | 필요 풀 |
    | --- | --- |
    | Lv1 → Lv2 | 10 |
    | Lv2 → Lv3 | 15 |
    | Lv3 → Lv4 | 20 |
    | Lv4 → Lv5 | 25 |
    | Lv5 → Lv6 | 30 |
    | Lv6 → Lv7 | 35 |
    | 이후 | +5씩 증가 |
7. 상점 가격 구간
    
    
    | 카테고리 | 가격 범위 | 목적 |
    | --- | --- | --- |
    | 소형 아이템 | 5 ~ 10 풀 | 즉시 구매 경험 |
    | 중형 아이템 | 15 ~ 25 풀 | 2~3일 목표 |
    | 대형 아이템 | 30 ~ 40 풀 | 장기 목표 |
    - 모자 : 5풀
    - 안경: 10풀
    - 옷: 15풀
