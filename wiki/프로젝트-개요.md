# 1. 프로젝트 개요

## 서비스 한 줄

**BiFriends** — 느린학습자(경계선 지능) 아동을 위한 학습·정서 지원 앱. AI 친구 **Leo(레오)** 와 대화하고, 공부방·친구랑(감정 학습)·부모 리포트를 제공합니다.

## 문제와 목표

| | |
|---|---|
| **대상** | 경계선 지능 아동 + 보호자 |
| **아이 화면** | 할 일·학습·Leo 채팅·친구랑·상점 |
| **부모 화면** | PIN 인증 후 주간 성장 리포트·챗 안전 신호 |

## 백엔드 역할

- Flutter 앱용 **REST API** (`/api/v1/**`)
- **PostgreSQL** — 회원, 학습, 채팅, 할 일, 리포트, 상점
- **Firebase** — Firestore(친구랑 세션), Storage(이미지)
- **bifriends-ai** 연동 — 채팅, 시나리오, 주간 배치

## 주요 도메인

| 도메인 | 기능 |
|--------|------|
| member / onboarding | Google 로그인, 온보딩, JWT |
| home / todo | 홈, 할 일, 출석·보상 |
| learning | 수학·국어 공부방 |
| chat | Leo 채팅 |
| mind | 친구랑 감정 시나리오 |
| report / safety | 부모 주간 리포트, 챗 안전 신호 |
| shop | 아이템 상점 |
| parent | 부모 PIN |
