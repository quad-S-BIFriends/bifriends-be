# 4. 나의 역할 — Backend (`bifriends-be`)

## 담당 범위

| 영역 | 상세 |
|------|------|
| **API 설계·구현** | `/api/v1` REST, Swagger 명세 정합 |
| **도메인 개발** | member, home, chat, learning, mind, report, shop, parent 등 |
| **AI 연동** | BE↔`bifriends-ai` HTTP, 내부 API(`X-Internal-Service`), 콜백 |
| **인프라 연동** | PostgreSQL, Firebase(Firestore/Storage), Docker 배포 |
| **운영·품질** | 버그 수정, 트랜잭션/성능 이슈, Flyway, VM 배포 |

## 주요 구현 기능

- Google OAuth → Firebase ID Token 검증 → **JWT** 발급
- Leo **채팅** 중계 (`ChatService` + AI 클라이언트)
- **친구랑** `/api/v1/mind/**` (시나리오 생성·세션 저장)
- **부모 주간 리포트** + 월요일 **배치 스케줄러**
- **챗 안전 신호** BE 콜백·조회
- 홈 **할 일 스케줄러**, 출석·보상 연동
