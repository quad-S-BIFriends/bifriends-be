## MVP 한계 (인지하고 있는 것들)

| 항목 | 현재 상태 | 향후 개선 |
|------|----------|----------|
| Refresh Token 무효화 | DB/Redis 미저장 → 탈취 시 만료까지 무효화 불가 | Redis 블랙리스트 |
| 스케줄러 분산 중복 실행 | 서버 2대 이상 시 AI 트리거 N배 발송 | ShedLock 또는 Spring Batch |
| `TodoScheduler.findAll()` | 전체 회원 메모리 로드 | Spring Batch Chunk 처리 |
| AI 콜백 미수신 | 재시도 전략 없음 → 해당 주 리포트 누락 | Dead Letter Queue 또는 재시도 스케줄러 |
